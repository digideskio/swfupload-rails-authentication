= swfupload-rails-authentication

Demo Rails 3.0.0.rc app showing SWFUpload working in tandem with restful-authentication, CSRF protection and paperclip.

== Usage
  
  gem install bundler --pre
  bundle install
  rake db:migrate
  rake db:seed
  rails s
